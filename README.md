

"# mini-program-movie" 

须在后台建立 movies, collection, user, review 四个数据库，使用other目录下的csv文件分别初始化数据库

只需要初始化 movies 并导入数据内容，
其他collection, user, review 三个数据库只需要在云数据库里创建文件夹，项目运行中会自动构建数据结构。
